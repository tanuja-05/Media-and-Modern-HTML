What is HTML5?

HTML5 is the latest version of the Hypertext Markup Language used to structure content on the web.

What is Semantic HTML5?

Semantic HTML5 is the use of HTML tags that convey meaning to both the browser and the developer.

Common Semantic Tags

  • <header> - Represents a container for introductory content or a set of navigational links.
  • <nav> - Defines a section of navigation links.
  • <main> - Specifies the main content of the document.
  • <footer> - Represents the footer of a section or a page.
